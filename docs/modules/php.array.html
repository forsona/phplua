Functions

array_keys (array, search_value, strict)
返回数组中部分的或所有的键名

Parameters:

  • array 一个数组，包含了要返回的键
  • search_value 如果指定了这个参数，只有包含这些值的键才会返回
  • strict 判断在搜索的时候是否该使用严格的比较（===）

Returns:

    array
array_change_key_case (input, case)
返回字符串键名全为小写或大写的数组

Parameters:

  • input 需要操作的数组
  • case 可以在这里用两个常量，CASEUPPER 或 CASELOWER（默认值）

Returns:

    array 返回一个键全是小写或者全是大写的数组；如果输入值（input）不是一个数组，那么返回false
array_chunk (input, size, preserve_keys)
将一个数组分割成多个

Parameters:

  • input 需要操作的数组
  • size 每个数组的单元数目
  • preserve_keys 设为 true，保留输入数组中原来的键名。如果你指定了false，那每个结果数组将用从1开始的新数字索引。默认值是false

Returns:

    array 得到的数组是一个多维数组中的单元1，每一维包含了 size 个元素
array_column (input, column_key, index_key)
返回数组中指定的一列

Parameters:

  • input 需要取出数组列的多维数组（或结果集）
  • column_key 需要返回值的列，它可以是索引数组的列索引，或者是关联数组的列的键。 也可以是nil，此时将返回整个数组（配合index_key参数来重置数组键的时候，非常管用）
  • index_key 作为返回数组的索引/键的列，它可以是该列的整数索引，或者字符串键值

Returns:

    array 从多维数组中返回单列数组
array_combine (keys, values)
创建一个数组，用一个数组的值作为其键名，另一个数组的值作为其值

Parameters:

  • keys 将被作为新数组的键。非法的值将会被转换为字符串类型（string）
  • values 将被作为数组的值

Returns:

    array 返回合并的array，如果两个数组的单元数不同则返回false
array_count_values (input)
统计数组中所有的值出现的次数

Parameters:

  • input 统计这个数组的值

Returns:

    array 返回一个关联数组，用 input 数组中的值作为键名，该值在数组中出现的次数作为值
array_key_exists (key, search)
检查给定的键名或索引是否存在于数组中

Parameters:

  • key 要检查的键
  • search 一个数组，包含待检查的键

Returns:

    array 成功时返回true， 或者在失败时返回false
array_sum (array)
计算数组中所有值的和

Parameters:

  • array 输入的数组

Returns:

    array 所有值的和以整数或浮点数的结果返回，除了数字格式的字符串和数字，其他被认作0
count (var, mode)
计算数组中的单元数目或对象中的属性个数

Parameters:

  • var 数组或者对象
  • mode 如果可选的 mode 参数设为 COUNT_RECURSIVE（或 1），count() 将递归地对数组计数。对计算多维数组的所有单元尤其有用。mode 的默认值是 0。count() 识别不了无限递归

Returns:

    number 返回 var 中的单元数目

Tables

array
预定义常量

Fields:

  • CASE_LOWER 用在array_change_key_case()中将数组的键名转换成小写字母。默认值
  • CASE_UPPER 用在array_change_key_case()中将数组的键名转换成大写字母
  • COUNT_NORMAL 用在 count() 中，不递归，默认值
  • COUNT_RECURSIVE 用在 count() 中递归计算数组元素

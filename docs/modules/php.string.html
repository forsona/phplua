defaultValue (delimiter, string, limit)
使用一个字符串分割另一个字符串

Parameters:

  • delimiter 边界上的分隔字符
  • string 输入的字符串
  • limit
    • 如果设置了 limit 参数并且是正数，则返回的数组包含最多 limit 个元素，而最后那个元素将包含 string 的剩余部分。
    • 如果 limit 参数是负数，则返回除了最后的 -limit 个元素外的所有元素。
    • 如果 limit 是 0，则会被当做 1

Returns:

    array 此函数返回由字符串组成的数组，每个元素都是 string 的一个子串，它们被字符串 delimiter 作为边界点分割出来
